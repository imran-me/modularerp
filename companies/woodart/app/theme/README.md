# Woodart Interiors — theme

Per-company Tailwind overrides on top of the shared design system
(`platform/design-system/`). This company's accent is `#6f9c1c`; the
router injects it at runtime as `--accent`, so every accent-derived colour
(buttons, chips, active nav, charts) tints to this company automatically — no
new colours are introduced here (R1).
